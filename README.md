# MatchMind
MatchMind is a full-stack TypeScript application for predicting La Liga match outcomes. The backend is built with Node.js and Express, integrating with the football-data.org API to sync real match data into a PostgreSQL database via Prisma ORM. A weighted recency-based algorithm processes historical match results to generate predictions, with an accuracy tracker measuring model performance over time. The frontend, built with Next.js and Tailwind CSS, surfaces predictions and team data through a clean, filterable UI. The project emphasizes modular architecture, type safety across the stack, clean API design, and practical data modeling — built end-to-end as a portfolio demonstration of full-stack engineering skills.
